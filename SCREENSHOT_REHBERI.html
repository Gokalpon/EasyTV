🛠️ Screenshot Nasıl Alınır? (Mac Olmadan — En Kolay Yol)

1 Chrome DevTools ile tarayıcıdan al
1. Chrome'da www/index.html aç
2. F12 → DevTools → Toggle Device Toolbar (Ctrl+Shift+M)
3. Cihaz: iPhone 14 Pro Max veya custom 430×932 seç
4. Zoom: %100 yap
5. DevTools → ⋮ → "Capture screenshot" veya "Capture full size screenshot"
6. PNG olarak kaydedilir — sonra 1290×2796'ya upscale et
2 Screenshot üzerine başlık ekle (Canva / Figma)
1. canva.com → Mobile App Screenshot şablonu aç
2. Screenshot'ı al, üstüne Türkçe/İngilizce başlık ekle
3. Export: PNG, 1290×2796 px

Alternatif: appshot.io veya screenshots.pro — otomatik çerçeve ekler
3 Mac varsa — Xcode Simulator (En İyi Kalite)
1. npx cap open ios → Xcode aç
2. Simulator: iPhone 16 Pro Max seç
3. Uygulamayı çalıştır
4. Simulator menüsü → File → Save Screen Shot (Cmd+S)
5. Otomatik 1320×2868 kaydeder
🚀 En hızlı yol: Chrome DevTools + Canva kombinasyonu. Mac olmadan 1-2 saatte 5 adet professional screenshot hazırlayabilirsin.
